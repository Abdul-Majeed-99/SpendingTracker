========================================
Project Description
========================================

SpendingTracker is a simple expense management system developed using Core Java and MySQL Database.

The purpose of this project is to help users manage and track their daily expenses in an organized way.
 The system allows users to add, update, delete, and view expense records using a graphical user interface.

This project was created for learning Object Oriented Programming concepts, Java GUI development, JDBC connectivity, and database integration.

========================================
Database Setup
========================================

1. Open XAMPP Control Panel
2. Start Apache and MySQL
3. Open phpMyAdmin
4. Create a new database

Example Database Name:
spendingdb

5. Import the SQL file from the "sql" folder
6. Make sure all required tables are created successfully

========================================
Java and JDBC Setup
========================================

1. Open the project in NetBeans IDE
2. Add MySQL JDBC Connector JAR file in project libraries
3. Check database connection settings in the Java files

Example Connection:

URL:
jdbc:mysql://localhost:3306/spendingdb

Username:
root

Password:
(leave empty if default XAMPP setup)

========================================
How to Run the Project
========================================

1. Open XAMPP and start MySQL
2. Open NetBeans IDE
3. Open the SpendingTracker project
4. Build the project
5. Run the main Java file
6. The application GUI will open
7. Use the system to manage expense records
